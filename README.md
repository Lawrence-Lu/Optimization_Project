# Optimization_Project

## 项目结构
`方法名.ipynb` 这样的后缀名为调用的 sample
- `data` 文件夹，存放数据
- `gen.py` 随机数生成
- `load_data.py` 加载指定数据
- `plot.py` 绘图相关
- `AGM.py` 加速梯度法
- `Newto_CG.py` 牛顿CG法
- `func_tools.py` 和损失函数相关的辅助函数（类似求梯度等）
- ...
